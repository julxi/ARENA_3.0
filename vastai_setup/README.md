To use this do the following (all in the `vastai_setup` folder):
1.  create `.env.vastai` and fill with `VASTAI_API_KEY=<your key>` and `VASTAI_INSTANCE_ID=<instance id>`
2. install `vastai_setup/requirements.txt`
3. run `setup.py`

This should setup this repo on the VM (except any necessary vs code extensions)
